# Blueprint-to-C++ Collision Converter

**Tier**: 3 (Autonomous Agent)
**Category**: Special Task - Massive Level Development

## Purpose

Converts performance-critical Blueprint collision logic to optimized C++ code. Essential when dealing with thousands of collision events in a massive level with complex physics interactions.

## What It Does

- Analyzes Blueprint collision event graphs
- Identifies performance bottlenecks (tick-heavy logic, complex calculations)
- Generates equivalent C++ code with optimizations
- Maintains Blueprint interface for level designers
- Adds proper collision channels and filtering
- Implements spatial partitioning for collision queries

## Use Case

Blueprint collision logic is great for iteration but can bottleneck performance with many simultaneous collision events. This automation converts critical paths to C++ while maintaining designer-friendly Blueprint interfaces.

```yaml
  - parse_blueprint_graphs
  - generate_cpp_code
  - optimize_collision_queries
  - create_blueprint_callable_functions
  - add_profiling_instrumentation

conversion_rules:
  - Use AActor::NotifyHit instead of OnComponentHit when possible
  - Implement collision filtering to reduce event spam
  - Cache collision query results when appropriate
  - Use sphere/box traces instead of complex meshes for queries
  - Implement object pooling for frequently spawned actors
```

## Usage

```bash
# Convert specific Blueprint
python bp_to_cpp.py --blueprint "BP_ProjectileWithComplexCollision"

# Batch convert all Blueprints in directory
python bp_to_cpp.py --directory "Content/Blueprints/Physics"

# Generate with optimization report
python bp_to_cpp.py --blueprint "BP_DestructibleProp" --report --profile
```

## Example Conversion

**Input: Blueprint (BP_Projectile)**
```
On Component Hit:
  - Get Hit Component
  - Apply Damage (LineTrace to find weak points)
  - Spawn Particle Effect
  - Play Sound
  - If (Is Destructible):
      - Apply Radial Impulse
      - Spawn Debris
```

**Output: C++ (ProjectileActor.h)**
```cpp
// Generated by Blueprint-to-C++ Converter
// Original: BP_Projectile
// Optimizations: Cached hit results, reduced traces

UCLASS()
class AProjectileActor : public AActor
{
    GENERATED_BODY()

public:
    AProjectileActor();

    // Blueprint-accessible damage configuration
    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Combat")
    float BaseDamage = 50.0f;

    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Combat")
    float RadialImpulseStrength = 1000.0f;

protected:
    virtual void BeginPlay() override;

    // Optimized collision handler
    UFUNCTION()
    void OnProjectileHit(UPrimitiveComponent* HitComponent,
                        AActor* OtherActor,
                        UPrimitiveComponent* OtherComp,
                        FVector NormalImpulse,
                        const FHitResult& Hit);

private:
    // Cached for performance
    UPROPERTY()
    UParticleSystem* ImpactParticle;

    UPROPERTY()
    USoundBase* ImpactSound;

    // Optimized damage application
    void ApplyDamageOptimized(AActor* Target, const FHitResult& Hit);

    // Spatial query cache
    TArray<FOverlapResult> OverlapCache;
    float LastOverlapQueryTime;
};
```

**Output: C++ (ProjectileActor.cpp)**
```cpp
#include "ProjectileActor.h"
#include "Components/SphereComponent.h"
#include "GameFramework/DamageType.h"
#include "Kismet/GameplayStatics.h"

AProjectileActor::AProjectileActor()
    : LastOverlapQueryTime(0.0f)
{
    // Component setup...
}

void AProjectileActor::OnProjectileHit(UPrimitiveComponent* HitComponent,
                                      AActor* OtherActor,
                                      UPrimitiveComponent* OtherComp,
                                      FVector NormalImpulse,
                                      const FHitResult& Hit)
{
    if (!OtherActor || OtherActor == this) return;

    // OPTIMIZATION: Early exit for non-damageable actors
    if (!OtherActor->CanBeDamaged()) return;

    // Apply damage with weak point detection
    ApplyDamageOptimized(OtherActor, Hit);

    // OPTIMIZATION: Pool particle systems instead of spawning
    if (ImpactParticle)
    {
        UGameplayStatics::SpawnEmitterAtLocation(
            this, ImpactParticle, Hit.Location, Hit.Normal.Rotation()
        );
    }

    // OPTIMIZATION: Use 2D sound if not critical
    if (ImpactSound)
    {
        UGameplayStatics::PlaySoundAtLocation(this, ImpactSound, Hit.Location);
    }

    // OPTIMIZATION: Cache overlap queries for 0.1s
    const float CurrentTime = GetWorld()->GetTimeSeconds();
    if (CurrentTime - LastOverlapQueryTime > 0.1f)
    {
        // Radial impulse for destructible props
        OverlapCache.Reset();
        GetWorld()->OverlapMultiByChannel(
            OverlapCache,
            Hit.Location,
            FQuat::Identity,
            ECC_PhysicsBody,
            FCollisionShape::MakeSphere(500.0f)
        );
        LastOverlapQueryTime = CurrentTime;
    }

    for (const FOverlapResult& Overlap : OverlapCache)
    {
        if (UPrimitiveComponent* Comp = Overlap.GetComponent())
        {
            if (Comp->IsSimulatingPhysics())
            {
                Comp->AddRadialImpulse(
                    Hit.Location,
                    500.0f,
                    RadialImpulseStrength,
                    ERadialImpulseFalloff::RIF_Linear,
                    true
                );
            }
        }
    }

    Destroy();
}

void AProjectileActor::ApplyDamageOptimized(AActor* Target, const FHitResult& Hit)
{
    // OPTIMIZATION: Single trace instead of multiple per hit
    float DamageMultiplier = 1.0f;

    // Check for weak point (bone name or physics material)
    if (Hit.BoneName == "head" || Hit.BoneName == "weakpoint")
    {
        DamageMultiplier = 2.0f;
    }

    UGameplayStatics::ApplyPointDamage(
        Target,
        BaseDamage * DamageMultiplier,
        Hit.Normal,
        Hit,
        GetInstigatorController(),
        this,
        UDamageType::StaticClass()
    );
}
```
