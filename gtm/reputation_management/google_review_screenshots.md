# Automate Google Review Screenshots (Public Pain Scraper)

## Objective

Find local businesses with unaddressed negative reviews, scrape the review using a screenshot tool, and pitch a reputation management solution using that visual proof.

## Tool Stack

- **Clay** - Workflow Orchestration
- **Zenrows** - Anti-bot Scraping & Screenshotting
- **Google Maps/Yelp** - Review Source
- **OpenAI** - Response Drafting

## Workflow Summary

This automation creates a reputation management lead generation system by:
1. Identifying businesses with negative reviews on Google Maps/Yelp
2. Using Zenrows to bypass anti-bot protections and capture screenshots
3. Extracting review content and sentiment
4. Generating contextual pitch emails with OpenAI
5. Using visual proof (screenshot) to demonstrate the problem in outreach
