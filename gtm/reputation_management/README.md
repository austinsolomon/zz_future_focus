# Reputation Management

**Purpose**: Monitor and manage online reputation, reviews, and brand perception.

## Use Cases

- **Review Monitoring**: Track G2, Capterra, Google reviews
- **Review Response Automation**: Auto-respond to reviews
- **Sentiment Analysis**: Analyze review sentiment trends
- **Screenshot Documentation**: Archive reviews for sales enablement

## Company Size Applications

- **SMB**: Basic review tracking (Google, Yelp)
- **Mid-Market**: Multi-platform monitoring + response automation
- **Enterprise**: Comprehensive brand monitoring + crisis detection
