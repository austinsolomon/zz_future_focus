# Lead Sequencing & Personalized Outreach

**Purpose**: Create and execute personalized outbound sequences across multiple channels.

## Use Cases

- **Multi-Channel Sequences**: Email → LinkedIn → Phone cadences
- **Personalized Email Generation**: AI-written emails based on research
- **Follow-up Automation**: Triggered follow-ups based on engagement
- **A/B Testing**: Test subject lines, messaging, timing

## Company Size Applications

- **SMB**: Simple 3-5 touch sequences (email-focused)
- **Mid-Market**: Multi-channel 7-10 touch sequences with personalization
- **Enterprise**: Complex multi-stakeholder campaigns (15+ touches)
