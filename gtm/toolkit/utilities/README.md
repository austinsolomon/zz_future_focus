# GTM Toolkit - Utilities

**Purpose**: Cross-cutting utilities and discovery tools for GTM automation.

---

## department_workflow_mapper/

**What it is**: Systematically discover high-ROI automation candidates for any GTM department.

**How to use**:
1. Copy pre-filled prompt for your department (Marketing, Sales, BDR, CS, RevOps, Finance)
2. Paste into Claude
3. Receive 10 ranked workflow candidates with ROI scores
4. Get ready-to-submit intake previews

**Files**:
- `README_workflow_discovery.md` - Full system overview
- `workflow_discovery_examples.md` - Pre-filled prompts for 6 departments
- `workflow_discovery_prompt.md` - Framework and methodology
- `workflow_discovery_quick_start.md` - Fill-in-the-blank template

**Use Case**: "I need to find automation opportunities in my Sales team"

**Time**: 5 minutes to generate 10 ranked candidates

---

## When to Use This

Use the workflow discovery system when:
- Starting a new automation initiative
- Prioritizing automation backlog
- Evaluating ROI across departments
- Generating intake submissions

This complements the 13 use-case folders by helping you **discover which use cases** to focus on first.
