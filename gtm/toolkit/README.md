# GTM/BDR Prospecting Toolkit

## Special Task: Niche Startup iPhone App Lead Generation
Finding startups on social media who need iPhone applications, focusing on unique verticals (influencers, luxury services, niche markets).

## Success Stories
- **Vegan Influencer**: 1M followers, needed branded recipe/meal planning app
- **Private Jet Rental**: Luxury service needed on-demand booking app

## Quick Start

Each automation directory contains:
- `README.md` - Detailed documentation
- `automation.yaml` - Automation configuration
- Implementation files (Python scripts, API integrations)
- Example data and outputs

## Integration Points

- **Social Media**: Twitter/X, LinkedIn, Instagram, TikTok
- **CRM**: Salesforce, HubSpot, Pipedrive
- **Enrichment**: Clearbit, Apollo, ZoomInfo
- **Communication**: Email (SendGrid), LinkedIn, Slack

## Tier Definitions

- **Tier 1**: Simple, deterministic automations (scrapers, templates)
- **Tier 2**: Context-aware with AI (personalized outreach, analysis)
- **Tier 3**: Autonomous agents with complex decision-making (lead qualification, scoring)
