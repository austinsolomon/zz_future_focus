# GTM Toolkit

**Purpose**: Cross-cutting utilities and reusable components for GTM automation.

---

## Folders

### utilities/
Cross-cutting discovery and helper tools:
- **department_workflow_mapper/** - Systematically discover automation opportunities

### commands/
Reusable command templates and scripts (placeholder for future use):
- CLI commands
- API call templates
- Data transformation scripts
- Testing utilities

---

## Relationship to Use Case Folders

**Use Case Folders** (`tam_accounts/`, `find_decision_makers/`, etc.):
- Complete automation examples
- Organized by business problem
- SMB/MM/ENT guidance

**Toolkit**:
- Utilities that span multiple use cases
- Discovery tools (workflow mapper)
- Reusable components (future commands)

---

## Quick Navigation

**Need to find automation opportunities?**
→ `utilities/department_workflow_mapper/`

**Need reusable commands?**
→ `commands/` (coming soon)

**Need automation examples?**
→ Go to use case folders (not toolkit)
