# GTM Toolkit - Commands

**Purpose**: Reusable command templates and scripts for GTM automation workflows.

---

## What Goes Here

This folder is for:
- **CLI commands** used across multiple GTM automations
- **API call templates** for common integrations
- **Data transformation scripts**
- **Testing utilities**

Examples:
- HubSpot API call templates
- Salesforce SOQL query builders
- Email validation scripts
- Lead scoring calculators

---

## How to Use

Commands here can be imported/referenced by automation examples in the use-case folders:
- `tam_accounts/`
- `find_decision_makers/`
- `lead_enrichment/`
- etc.

---

## Currently Empty

This folder is a placeholder for future command utilities. As common patterns emerge across the use-case examples, extract them here for reusability.
