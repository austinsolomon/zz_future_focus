# Automate Customer Feedback Analysis (Competitor Poaching)

## Objective

Scrape a competitor's public feedback board to find customers complaining about specific features, identifying high-pain duplicates to frame your product as the solution.

## Tool Stack

- **Clay** - Workflow Orchestration
- **Zenrows** - Complex Scraper for Lazy-Loading Sites
- **Competitor Boards** - e.g., Canny, ProductBoard
- **AI** - Semantic Duplicate Detection

## Workflow Summary

This automation turns competitor complaints into qualified leads by:
1. Scraping public feedback boards (Canny, etc.) using Zenrows
2. Handling lazy-loading and infinite scroll to capture all feedback
3. Extracting feature requests and pain points
4. Using AI to detect semantic duplicates (multiple people asking for same thing)
5. Identifying high-pain features where your product excels
6. Building targeted outreach lists of frustrated users ready to switch
