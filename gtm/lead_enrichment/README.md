# Lead Enrichment

**Purpose**: Augment lead/contact records with additional data to improve targeting and personalization.

## Use Cases

- **Firmographic Enrichment**: Company size, revenue, industry, location
- **Technographic Enrichment**: Tech stack, tools, integrations
- **Contact Data**: Email, phone, LinkedIn, job title
- **CRM Data Hygiene**: Keep records up-to-date and complete

## Company Size Applications

- **SMB**: Basic enrichment (company + email verification)
- **Mid-Market**: Comprehensive enrichment (tech stack + intent signals)
- **Enterprise**: Deep enrichment + custom data sources

## Examples in This Folder

- **lead_research_enrichment.json** (Tier 2) - AI-enhanced research workflow
- **two_way_crm_sync.md** (Shelf) - Bidirectional CRM enrichment
- **clay_prospecting_automation.md** (Shelf) - Clay-based enrichment pipeline
