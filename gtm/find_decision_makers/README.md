# Find Decision Makers at Accounts

**Purpose**: Identify and gather intelligence on key decision-makers and influencers at target accounts.

## Use Cases

- **Contact Discovery**: Find VP/C-level decision-makers at target accounts
- **Org Chart Mapping**: Understand reporting structures and stakeholders
- **Multi-Threading**: Identify multiple contacts per account
- **Contact Enrichment**: Get email, LinkedIn, phone for outreach

## Company Size Applications

- **SMB**: Single decision-maker focus (usually founder/CEO)
- **Mid-Market**: 2-3 stakeholders (VP Sales, VP Ops, CTO)
- **Enterprise**: Multi-threaded approach (5+ stakeholders across departments)
