# Second Brain (BR2) Toolkit

## Special Task: Implementing Second Brain with Claude Code + Obsidian
Implementing Thiago Forte's Second Brain methodology using Claude Code, Obsidian, and maximum automation for knowledge management and personal productivity.

## Core Methodology: PARA

**P**rojects - Short-term efforts with deadlines
**A**reas - Long-term responsibilities
**R**esources - Topics of interest
**A**rchives - Inactive items from other categories

## Integration Points

- **Obsidian**: Primary knowledge base
- **Claude Code**: AI-powered processing and analysis
- **File System**: PARA folder structure
- **Git**: Version control for notes
- **Dataview**: Obsidian query plugin
- **Templater**: Dynamic note templates

## Tier Definitions

- **Tier 1**: Simple, deterministic automations (folder creation, templates)
- **Tier 2**: Context-aware with AI (tagging, summarization)
- **Tier 3**: Autonomous agents with complex decision-making (linking, analysis)
