# BR2 - Tier 6 - Autonomous Knowledge Curator

**Autonomous Knowledge Management**: Learns your note patterns and autonomously curates connections without human intervention.

## Installation
```bash
cd br2/tier_6/
python tier_6_toy_autonomous_br2_knowledge_curator.py
```

## Tier 6: Personalized Learning
- Suggests connections autonomously
- Learns from accept/reject decisions
- Adapts to your knowledge patterns
- Improves relevance over time

**Feedback Loop**: Suggest → Accept/Reject → Learn Patterns → Better Suggestions → Personalized System
